## 贡献指南 / Contributing

### 安装 / Setup

```bash
# 拉取项目 / Clone repo
$ git clone https://github.com/PatchyVideo/Touhou-Vote.git
$ cd patchyvideo-vue

# 安装依赖 / Install dependences
$ npm i
```

### 调试 / Development

```bash
# 启动本地调试 / Start a dev server
$ npm run dev
```

### 构建 / Production

```bash
# 构建 & 运行 / Build & Start the project
$ npm run build
$ npm run start
```
